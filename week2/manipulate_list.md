# Truy cập phần tử

```python
numbers = [1, 2, 3, 4, 5]
first = numbers[0]    # 1
last = numbers[-1]    # 5
```

# Cắt list (slicing)

```python
numbers[1:3]    # [2, 3]
numbers[:3]     # [1, 2, 3]
numbers[2:]     # [3, 4, 5]
```

# Thêm/xóa phần tử:

```python
numbers.append(6)     # Thêm vào cuối
numbers.insert(0, 0)  # Thêm vào vị trí
numbers.remove(3)     # Xóa phần tử
```
