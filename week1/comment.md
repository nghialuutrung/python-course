# Single-line comments:

```python
# Đây là comment một dòng
x = 10  # Comment sau code
```

# Multi-line comments:

```python
"""
Đây là comment nhiều dòng
Thường dùng cho docstring
hoặc comment dài
"""
```

# Docstrings:

```python
def add(a, b):
    """
    Hàm cộng hai số.
    
    Args:
        a: Số thứ nhất
        b: Số thứ hai
        
    Returns:
        Tổng hai số
    """
    return a + b
```
